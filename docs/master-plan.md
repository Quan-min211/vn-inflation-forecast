# Master Plan

## Scope

The project is reduced to one defensible analytical track:

> FEVD-based proof that Vietnam's inflation is mainly inertial.

## Phases

1. **Data Cleaning**
   - Load Excel.
   - Rename typo column.
   - Interpolate missing monetary variables.
2. **Model Readiness**
   - ADF test.
   - Difference non-stationary variables.
   - VIF check.
3. **VAR and FEVD**
   - Fit VAR(1).
   - Forecast short horizon for sanity check.
   - Compute FEVD for 10 periods.
4. **Conclusion**
   - Compare CPI self-impact vs external factor contribution.
   - State policy implication: manage inflation expectations first.
